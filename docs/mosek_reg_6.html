Introduction

Similar to previous simulations, where correlated null \(z\) scores are fitted by \(10\) Gaussian derivatives with regularization, we are now fitting data sets simulated with correlated noise and true signals by both ASH and Gaussian derivatives using the same regularization and see what happens.

The study is almost exactly the same as the previous one, the single only change being that this time, the correlated noise is heteroskedastic. That is, \(\hat s_j\) are not constant; in this simulation, they are simulated such that \(\log\hat s_j \sim \text{Uniform}\left[\log0.1, \log10\right]\).

library(ashr)
source("../code/gdash.R")
